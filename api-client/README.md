# SAT Tutor AI Client

Python client for the SAT tutoring AI service.

## What's in this folder
- `sat_tutor_client.py`
- `.env.example`
- `requirements.txt`

## Quick Setup

1. Install dependencies:
   pip install -r requirements.txt
2. Set up your credentials:
cp .env.example .env
# Then edit .env with your actual values
3. Test it:
python sat_tutor_client.py
